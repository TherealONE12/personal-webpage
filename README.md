## 📖 Overview

This repository hosts my Personal Webpage a website designed for me to learn a bit more about html and CSS and as a fun little side projekt for displaying my skills, experience, and projects. It provides a nice interface for visitors to learn about my professional journey and explore various works I've completed.

## ✨ Features

-   **👤 Personal Introduction:** A dedicated section to introduce myself and highlight key professional attributes.
-   **🚀 Project Showcase:** Detailed sections for showcasing completed projects, including:
    -   `EZ-Admin`: An administrative panel for updateing any .deb based system using apt.
    -   `GetToSpace`: A space-themed game, with both current versions (`V1`, `V2`).
-   **🎨 Custom Styling:** A clean and modern user interface implemented with custom CSS (That was really hard...).

## 🚀 Quick Start

To view this personal webpage locally, simply clone the repository and open the main HTML file in your web browser.

### Prerequisites
-   A  web browser (e.g. Firefox, Safari, Ecosia, Duckduckgo,...).

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/TherealONE12/personal-webpage.git
    cd personal-webpage
    ```

2.  **Open in Browser**
    Simply open the `index.html` file in your preferred web browser.
    ```bash
    # For macOS
    open index.html

    # For Windows
    start index.html

    # For Linux (using xdg-open, may vary)
    xdg-open index.html
    ```
    Alternatively, for a more robust local development experience, you can use a simple static server:
    ```bash
    # Using Python (if installed)
    python -m http.server 8000
    ```
    Then, open your browser and visit `http://localhost:8000`.

## 🚀 Deployment

This project is a static website and can be easily deployed to any static site hosting service.

-   **GitHub Pages**: The simplest way to deploy this project is via [GitHub Pages](https://pages.github.com/). Since the `index.html` is in the root, it will be automatically served once GitHub Pages is enabled for the `main` branch of the repository.
